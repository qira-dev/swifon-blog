- Admin can change (rename) any existing category at any time.
- Changes should automatically update across the entire website (menus, pages, URLs, posts).

2. Add New Categories:
- Admin can create unlimited new categories.
- Each category should have:
  - Name
  - Slug (SEO-friendly URL)
  - Description
  - Icon or image (optional)

3. Delete Categories:
- Admin can delete categories safely.
- Before deleting, allow reassignment of posts/products to another category.

4. Category Hierarchy:
- Support parent and sub-categories (nested categories).

5. SEO Settings:
- Each category should have:
  - Meta title
  - Meta description
  - SEO slug (editable)

6. Auto Page Generation:
- Each category should automatically create a dedicated landing page.
- Page should list all related posts, reviews, and comparisons.

7. Multi-language Support:
- Category names and descriptions can be edited per language.

8. Drag & Sort:
- Admin can reorder categories (drag & drop).

9. Visibility Control:
- Option to show/hide categories from frontend menu.

Goal:
Allow full control over categories from admin panel without touching code.

অ্যাডমিন প্যানেল কাস্টমাইজেশন: অ্যাডমিন প্যানেলে একটি ক্যাটাগরি ম্যানেজমেন্ট সিস্টেম থাকতে হবে, যেখানে আমি সহজে ক্যাটাগরি নাম পরিবর্তন, নতুন ক্যাটাগরি যোগ, এবং ক্যাটাগরি মুছে ফেলতে পারব। ক্যাটাগরি হায়ারার্কি (প্যারেন্ট ও সাব-ক্যাটাগরি) সমর্থন থাকতে হবে, এবং প্রতিটি ক্যাটাগরির জন্য SEO সেটিংস (মেটা টাইটেল, ডেসক্রিপশন) এডিট করা যাবে।

প্রোডাক্ট ও টুলস সিস্টেম: প্রোডাক্ট এবং টুল সম্পর্কিত তথ্য আলাদা টেবিলে থাকবে, যেখানে প্রতিটি প্রোডাক্টের ছবি, নাম, দাম, রিভিউ ইত্যাদি থাকবে। পোস্টের সাথে প্রোডাক্ট যুক্ত করার অপশন থাকবে, যাতে পোস্টে প্রোডাক্ট রিভিউ বা তুলনা দেখানো যায়।
